Python

1.How did it come to be made?

  • A programming language created in 1972 by Dennis Rich[3] of Bell Labs.
  • In 1972, Dennis Rich of the same Bell Laboratory finally created the language "C", which inherited the characteristics of the B language to control the PDP-11 computer.

2.Feature

  • Grammar to simplify and unify grammatical structures as much as possible
  • One of Python's biggest features. Python provides a powerful feature called repeatable objects.
  • Python does not have a primitive type, and everything is treated as an object.

3.Advantages and Disadvantages of Python

  1. advantages
    • High productivity
    • Quick feedback on grammar unification
    • Science/Engineering Friendly
    • an all-around language
  2. disadvantages
    • INon-Multi-Threading Issues
    • Messy package management and backward compatibility errors
