C language


1.How did it come to be made?

  • A programming language created in 1972 by Dennis Rich[3] of Bell Labs.
  • In 1972, Dennis Rich of the same Bell Laboratory finally created the language "C", which inherited the characteristics of the B language to control the PDP-11 computer.

2.The spirit of the C language

  • Trust the programmer
  • Don't prevent the programmer from doing what needs to be done
  • Keep the language small and simple
  • Provide only one way to do an operation
  • Make it fast, even if it is not guaranteed to be portable

3.Advantages and Disadvantages of C Language

  1. advantages
    • Very Fast speed
    • The binary size is small.
  2. disadvantages
    • It is a difficult language for junior programmers who ignore all functions that are likely to take any overhead on the hardware (so that programmers have to pay attention to them all).
